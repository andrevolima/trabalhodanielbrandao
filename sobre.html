    CSS (Cascading Style Sheets) é uma linguagem de folha de estilo usada para definir a aparência e o layout de um documento HTML. Ele permite que os desenvolvedores web controlem a apresentação visual dos elementos de uma página da web, incluindo fontes, cores, tamanhos, posicionamento e muito mais.

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar e estruturar o conteúdo de uma página da web. É a base de todos os sites na internet e é interpretado pelos navegadores para exibir o conteúdo de uma página da web, incluindo texto, imagens, links, formulários e outros elementos

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, usada principalmente para criar interatividade em páginas da web. Ele é executado no lado do cliente, ou seja, no navegador do usuário, e é uma das tecnologias fundamentais da web moderna.
